 K nested loops - O(n^k).
 b recursive calls per level and has L Levels - O(b^L) complexity.
 Binary Search over a range [1..n] has O(Log(n)) complexity
 DP Algorithm or an iterative routine over a 2D n x n  matrix in O(k) per cell runs in O(k x n^2).

You want to pick the current top 10 pages with the highest page ranks.

2. Use a Priority Queue

3. Use QuickSelect Algorithm

Ans - Priority Queue will help get the max rank quickly -
